# My Portfolio

A personal portfolio website showcasing my projects, skills, and web development expertise. It features interactive , responsive and modern design.

## Features

- **Projects Section:** Project title, description, skills.
- **Skills Section:** Highlights technical skills with badges or tags.
- **Responsive Design:** Works seamlessly across desktops, tablets, and mobile devices.
- **Contact Section:** Easy access to email, LinkedIn, and GitHub links.
- **Modern UI:** Clean, professional design with customizable theme.

## Technologies Used

- **Frontend:** React, CSS Modules
- **Utilities:** JavaScript, HTML5, CSS3
